## 短い説明文（132文字以内）

```
YouTube・Twitterなど好きなサイトの1日の利用時間を制限。上限超過で自動ブロック。カスタムサイト追加、統計グラフ表示対応。
```

---

## 詳細説明文

```
Productivity Guard は、SNS や動画サイトへの時間を使いすぎないよう守るための Chrome 拡張機能です。

■ 主な機能

【時間制限・ブロック】
・YouTube・Twitter/X をデフォルトで監視
・1日の利用上限（分）をサイトごとに自由に設定
・上限に達すると自動でブロック画面に切替え
・ブロック後も「5分だけ延長」バイパス機能付き

【カスタムサイト追加】
・Instagram、Reddit、TikTok など任意のサイトを URL で追加可能
・追加したサイトも同様に時間制限・ブロックの対象になる
・不要になったサイトはワンクリックで削除

【開く前に確認ダイアログ】
・対象サイトを開く前に「本当に開きますか？」と問いかけるオーバーレイを表示
・無意識のアクセスを防ぎ、意識的な行動を促す
・ON/OFF 切替可能
・「今日だけOFF」で当日中だけスキップ、翌日に自動復帰

【統計グラフ】
・過去 7日 / 30日の利用時間を棒グラフで可視化
・上限ラインを重ねて表示し、超過した日が一目でわかる
・カスタムサイトの統計も自動で追加

■ プライバシーについて
すべてのデータはお使いの端末のみに保存されます。外部サーバーへの送信は一切行いません。

■ 使い方
1. 拡張機能アイコンをクリックしてポップアップを開く
2. 「サイト管理」に URL を入力して監視したいサイトを追加
3. 各サイトの上限時間を設定して「保存」
4. あとは拡張機能が自動で管理します
```

### PNG 変換方法

**方法1: ブラウザ経由**
1. `promo/preview.html` をブラウザで開く
2. DevTools → `promo-440x280.svg` の要素を右クリック → 「名前を付けて保存」

**方法2: コマンドライン（Inkscape）**
```bash
inkscape promo-440x280.svg --export-png=promo-440x280.png --export-width=440 --export-height=280
```

**方法3: コマンドライン（rsvg-convert）**
```bash
rsvg-convert -w 440 -h 280 promo-440x280.svg -o promo-440x280.png
```
